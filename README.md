# Human Foot Biomechanics Model

## About the Project

This project presents a four-degree-of-freedom (4-DOF) mathematical model of the human foot developed as part of my academic work. The objective was to study the dynamic behaviour of the foot by representing the ankle, subtalar joint, toe joint and longitudinal arch using a state-space model.

Different foot conditions were analysed by modifying the stiffness and damping parameters while keeping the model structure unchanged. MATLAB was used to simulate and compare the response of each condition.

---

## Foot Conditions Analysed

- Healthy Foot
- Flat Foot
- High Arch Foot
- Weak Plantar Fascia
- Ankle Instability
- Athlete (Rigid) Foot

---

## Outputs Studied

The model predicts:

- Ankle rotation (θa)
- Subtalar rotation (θs)
- Toe rotation (θt)
- Longitudinal arch deformation (h)
